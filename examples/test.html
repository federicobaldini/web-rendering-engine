Web Rendering Engine
A toy browser engine built step by step in Rust
01 HTML Parser
Tokenizes raw HTML markup and builds a DOM tree with error recovery
02 CSS Parser
Parses selectors and declarations and resolves the cascade
03 Layout Engine
Computes block box dimensions, margins, borders and positions
04 Painter
Rasterizes display list items with Porter-Duff alpha compositing
Rendered to 800x600 PNG via ab_glyph and the Rust image crate
